# Agent context: Obsidian vaults

## What this repo is

- **Two Obsidian vaults** in one repo. You will help edit and maintain them.
- **D&D campaign**: Notes, session logs, characters, and worldbuilding.
- **Vault ownership**:
  - **Campaign** — mine; edit freely (session notes, characters, items, locations, etc.).
  - **D&D 5e 2024 Rules** — downloaded reference (5e rules, statblocks, monsters); prefer not to change unless I ask.

## Vault layout (expected)

- `Campaign/` — main campaign vault (notes, `.obsidian/` config, plugins, snippets).
- `D&D 5e 2024 Rules/` — reference vault; treat as read-only unless instructed.

## Conventions when editing

- **Links**: Use Obsidian `[[Wiki-style links]]` for note names; avoid raw paths when linking between notes.
- **Files**: Prefer `.md` for content. Don’t delete or rename files in the reference vault.
- **Language**: Match the language of the note (e.g. Spanish in campaign notes if that’s how they’re written).
- **Obsidian config**: `.obsidian/` holds settings, plugins, themes, snippets; change only when asked or when fixing something we’re working on.
